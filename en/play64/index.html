Play64

Play64 is a fantasy console created for "The Tool Jam 5". It is a game engine designed to develop titles with an aesthetic reminiscent of the classic Atari 2600.

Key Features

  • Controls: It features a D-pad and two action buttons.
  • Graphics: A 64x64 pixel display capable of rendering four shades of purple.
  • Programming: Games are written in Lua 5.1.
  • Implementation: The console is built using BlitzMax NG and the Raylib graphics library.

Control Mapping

  • UP: UP ARROW and W keys
  • DOWN: DOWN ARROW and S keys
  • LEFT: LEFT ARROW and A keys
  • RIGHT: RIGHT ARROW and D keys
  • Button A: Z and N keys
  • Button B: X and M keys

Additional Commands

  • ESC: Closes the application.
  • R: Resets the console.
  • F: Toggles fullscreen mode.
  • P: Pauses the game.
