Gio's Static Site Generator

A modern and intuitive static site generator, developed in Python. It stands out for its textual interface (TUI) based on Textual, which makes managing the site simple and direct, without the need for complex commands.

Features

  • Markdown Processing: Converts Markdown files into HTML pages, using metadata defined in the frontmatter.
  • Customizable Themes: Offers flexible templates with Jinja2 to personalize the site's look.
  • Automatic Galleries: Generates photo galleries complete with thumbnails and an integrated viewer.
  • Blog Management: Allows you to organize content into articles, ordering them chronologically.
  • Automatic Navigation: Automatically creates sidebars and breadcrumbs for intuitive navigation.
  • Multilingual: Supports Italian and English.
