Voxel64

A simple voxelspace renderer for the LÖVE framework, which simulates 3D flight over terrains using heightmaps. This project is based on voxelspace.

Features

  • Procedural Terrain Rendering: Dynamically creates terrain using height and color maps.
  • Flight Controls: Allows for free navigation.
  • Multiple Maps: The project includes several predefined terrain maps.
  • Basic HUD: An on-screen interface displays key information.

Requirements and Running

  • Requires the LÖVE 2D game engine.
  • To Run: Execute the game from the terminal with the command: love /path/to/voxel64-master

Controls

  • W/A/S/D: Move forward/left/backward/right
  • Up/Down Arrows: Increase/decrease altitude
  • Numbers 1-5: Change map
  • H: Show/hide HUD
  • Esc: Exit the game
