Luapydoc

Luapydoc is a Python-based documentation generator for Lua codebases. It parses LDoc-style documentation comments in Lua source files and generates a comprehensive, navigable HTML documentation website with syntax highlighting, search functionality, and more.

Repository

Output Example

Features

  • Parses LDoc documentation comments (@param, @return, @usage, etc.)
  • Supports functions, variables, tables, and types
  • Generates responsive HTML pages with a dark theme
  • Syntax highlighting for Lua code using Pygments
  • Tree-based sidebar navigation for modules, classes, and functions
  • Full-text search with an index
  • Responsive design for mobile and desktop
